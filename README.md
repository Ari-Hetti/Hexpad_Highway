# BOM
- 1 Seeed XIAO RP2040
- 6 MX-Style switches
- 6 Blank DSA keycaps
- 1 EC11 Rotary encoders
- 1 0.91 inch OLED display
- 6 SK6812 MINI-E LEDs
- 4 M3x16mm screws
- 4 M3x5mx4mm heatset inserts
